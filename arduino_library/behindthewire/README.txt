Copy the entire folder to your Arduino libraries folder

The directory structure should look like the following

C:\arduino-1.0\libraries\behindthewire\
C:\arduino-1.0\libraries\behindthewire\behindthewire.h 
C:\arduino-1.0\libraries\behindthewire\behindthewire.txt
C:\arduino-1.0\libraries\behindthewire\README.txt <-- This file
C:\arduino-1.0\libraries\behindthewire\examples\
C:\arduino-1.0\libraries\behindthewire\examples\FoggyDayTakeoff
C:\arduino-1.0\libraries\behindthewire\examples\FoggyDayTakeoff\FoggyDayTakeoff.ino
C:\arduino-1.0\libraries\behindthewire\examples\PassengerReadingLight
C:\arduino-1.0\libraries\behindthewire\examples\PassengerReadingLight\PassengerReadingLight.ino

behindthewire.h
---------------

Contains the definitions of the inputs and outputs connected on the BtW box.

behindthewire.h
---------------

Contains the keywords used by the btw library to allow correct syntax highlighting.

\examples\
-----------

Will contain the source code for the exercises as written on the worksheets.
