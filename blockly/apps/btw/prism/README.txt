## Prism
Prism is a lightweight, extensible syntax highlighter, built with modern web standards in mind.

### Changes to original source code
The source code can be customized on their website and this version has been based on the following configuration:

http://prismjs.com/download.html?themes=prism&languages=clike+c+cpp

The CSS file has been modified to fit the colour scheme of BlocklyDuino for Behind the Wire UI.
The JavaScript file has been modified to add keywords specific for the Arduino code.
